# AI-Powered QA Quality Platform

Modern QA automation platform demonstrating how AI can enhance software quality engineering.

## Features

- AI-generated API tests
- API automation with Playwright
- Contract testing
- Visual regression testing
- Performance testing with k6
- CI/CD Quality Gates
- Monitoring dashboards with Grafana
- Dockerized environment

## Tech Stack

Playwright  
Node.js  
OpenAI  
k6  
Grafana  
Prometheus  
Docker  
GitHub Actions

## Run locally

Install dependencies

npm install

Generate AI tests

npm run ai-tests

Run automation

npm test

Run performance tests

npm run performance

Start monitoring

npm run monitoring
